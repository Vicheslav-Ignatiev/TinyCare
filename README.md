# TinyCare
TinyCare is a web application for tracking newborn daily care: feeding times, intervals, diaper changes, intake amounts, and end-of-day analytics with efficiency insights.

## Features
- Feeding time tracking
- Feeding interval timers
- Diaper change logging
- Intake amount tracking
- End-of-day statistics and efficiency overview

## Tech Stack
- React
- Node.js
- MongoDB
- Docker
- Nginx

## Project Structure
/frontend   # React application
/backend    # Node.js API
/infra      # Docker & Nginx configuration
